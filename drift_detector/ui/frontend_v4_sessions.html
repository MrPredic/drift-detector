DriftDetector — How It Works

What is Drift Detection?

Drift detection monitors behavioral changes in LLM agents. When an agent starts producing inconsistent, repetitive, or degraded outputs, DriftDetector catches it before production impact.

5 Drift Signals

👻 Ghost Lexicon: Detects vocabulary shrinkage — when agent loses precision/domain-specific terms. High = vocabulary collapse.
🔄 Behavior Shift: Tracks tool usage pattern changes. High = agent changing which tools it uses.
🤝 Agreement: Measures output clarity/consistency. Low agreement = agent becoming vague/contradictory.
🔁 Loop Detection: Identifies repetitive action sequences. High = agent stuck in loops (retrying same thing).
Stagnation: Detects identical output repetition. High = agent producing same response repeatedly.

Scoring Scale

🟢 0.0-0.4 = Healthy behavior

🟡 0.4-0.6 = Elevated drift (monitor)

🔴 0.6+ = High drift (action recommended)
