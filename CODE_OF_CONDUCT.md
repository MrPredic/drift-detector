# Code of Conduct

## Our Commitment

We are committed to providing a welcoming and inspiring community for all. We value diverse perspectives and experiences.

## Our Standards

Examples of behavior that creates a positive environment:
- Being respectful and inclusive
- Using welcoming language
- Gracefully accepting constructive criticism
- Focusing on what's best for the community
- Showing empathy for others

Examples of unacceptable behavior:
- Harassment, discrimination, or intimidation
- Trolling, insulting comments, or ad hominem attacks
- Deliberately spreading misinformation
- Any illegal activity

## Enforcement

Instances of unacceptable behavior should be reported to the maintainers. All complaints will be reviewed and investigated. We are committed to maintaining a safe and welcoming space for all.

## Scope

This Code of Conduct applies to:
- GitHub issues and pull requests
- Project communications
- Any official community spaces

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/).
